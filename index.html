The highlights of my work

  • tjfs (late 2015)

    During my stay at University of New Orleans we developed a high-availability distributed file system based on the original Google FS design. The file system is written completely in Java.

  • Lumines (2015)

    JavaScript implementation of the legendary PSP game that you can try out on this page. It’s based on the bleeding-edge JS technologies like React.js, Flux and Immutable.js.

  • SEE | Matouš Roskovec – seewordpower.cz (2015)

    Coding and MODx Revolution backend for the personal web page of my cousin, a copywriter. The page is Bootstrap based, fully responsive and fairly small despite it is Retina ready, has a custom font and contains lots of graphical elements.

  • TellyPal (2014)

    My first bigger JavaScript project, with Node.js backend and React.js/Flux frontend. TellyPal allows you to easily manage and download new episodes of your favorite TV shows. It works as a thin layer over The TVDB, Torrentz and Deluge BitTorrent Client.

  • Bambule Android application (2014)

    A simple Android application (written in Java) that I developed together with my schoolmate Katka Nevolová as a school project for Alltoys. It offers some basic functionality for the customers of the Bambule retail stores, such as locating the nearest store and accessing club member profile.

  • Factor (2013-2014)

    Huge B2B e-shop that we developed in Alltoys using PHP/Nette. Factor was refined and polished to perfection with the specific needs of the wholesale environment in mind.

  • Túúdl (2012-2013)

    This was my bachelor’s thesis. It’s an application for high school e-learning, developed on Google App Engine platform, with Java/Spring MVC backend and HTML/JavaScript frontend. The point of the work was to face the challenges imposed by the cloud environment and a non-relational database.

  • SEM ČR website (2012)

    Complete website for a non-profit organisation: design, coding and MODx Evolution backend. It shows my humble Gimp painting skills. I also developed a custom module in PHP for managing the organisation agenda (members, fees…).

  • Svoboda.info (2010-2014)

    A huge local news portal that I designed and coded from scratch. It’s built on top of MODx Evolution but it contains lots of custom modifications and modules (i. e. football results of local contests). As the website grew, it reached the limits of the framework which required invasive hacking of the CMS core. As of July 2015, the website is maintained by an external company (with a proprietary CMS) but the design and functionality remains from 90% the same.
